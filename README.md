# 知育アプリ

## 概要

このアプリは、幼児向けの知育コンテンツを提供する Android アプリです。

## 機能一覧

### 基本機能

#### お知らせ

アプリ起動時にお知らせ内容を動的に表示する。

#### 強制アップデート

アプリバージョンが古い場合、アプリの使用をブロックし、アップデートを促す。

#### 利用規約同意

- 初回起動時に利用規約・プライバシーポリシーへの同意を求める。
- 規約更新時には再同意を促す。

### コンテンツ機能

#### 車両

- 数種類のカラフルな車両が動く。

- 車両をホールドすると、停車し、放置すると渋滞する。

#### 打楽器

- ボタンをタップすると打楽器が鳴る。
- 数種類の打楽器が存在する。

#### 魚群

- カラフルな魚群を表示する。

- 魚群の動きに[ボイドアルゴリズム](https://ja.wikipedia.org/wiki/%E3%83%9C%E3%82%A4%E3%83%89_(%E4%BA%BA%E5%B7%A5%E7%94%9F%E5%91%BD))を使用する。
- 画面をタップすると、魚はタップされた位置から離れる。

#### サインポール 

サインポールを再現する。

- 再生・一時停止できる。
- 回転速度を変更できる。
- 色を変更できる。

#### 万華鏡

万華鏡を再現する。

- 万華鏡を回転できる。
- 万華鏡に使用する画像を選択できる。
