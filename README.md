## こだわりポイント

### 直感的な UI

- 文字が読めなくても遊べるよう、大きなアイコンカードで各コンテンツへナビゲーション。
- ボタンは子供の指でもタップしやすいサイズに設計。

### 誤操作防止

- **画面ピン留め機能**により、子供が誤って他のアプリに切り替えることを防止。
- システムバーを非表示にするイマーシブモードで、誤操作による中断を排除。

### ポジティブなフィードバック

- タップやホールドに対して視覚的・触覚的（バイブレーション）なフィードバックを即座に返し、「自分の操作で変化が起きた！」という達成感を演出。

### 安全性

- 広告を一切排除。
- 外部サイトへの遷移なし。
- 利用規約・プライバシーポリシーへの同意フローを実装し、保護者への透明性を確保。

## 機能一覧

| 機能 | 子供へのメリット | 実装の工夫 |
| --- | --- | --- |
| 車両 | 因果関係の理解（触ると止まる、離すと動く） | カラフルな車両アニメーション。ホールドで停車し、放置すると渋滞が発生 |
| 打楽器 | リズム感・音感を育む | 8 種類の打楽器を色分けしたボタンで配置。タップで即座に音が鳴る |
| 魚群 | 自然界の動きへの興味を刺激 | ボイドアルゴリズムによるリアルな群れの動き。タップで魚が逃げる |
| サインポール | 色彩感覚・観察力を養う | OpenGL ES で描画。回転速度・色の変更が可能 |
| 万華鏡 | 創造性・美的感覚を育む | Vulkan による GPU アクセラレーション描画。画像選択・回転操作が可能 |

### 基本機能

| 機能 | 説明 |
| --- | --- |
| お知らせ | Firebase Remote Config で動的に表示。既読管理付き |
| 強制アップデート | 古いバージョンの使用をブロックし、アップデートを促す |
| 利用規約同意 | 初回起動時に同意を求め、規約更新時には再同意を促す |
