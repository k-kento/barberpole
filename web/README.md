# Barberpole Web

A web application for Barberpole.

## Local Development

To run the web application locally, you need to have the Firebase CLI installed.

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the local development server:
    ```bash
    firebase emulators:start
    ```

## Deployment

To deploy the web application to Firebase, run the following command:

```bash
firebase deploy
```
